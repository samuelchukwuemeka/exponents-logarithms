(2.) NYSED The table below gives the relationship between x and y

x 1 2 3 4 5
y 4.2 33.5 113.1 268.1 523.6

Use exponential regression to find an equation for y as a function of x, rounding all values to the nearest hundredth.

Using this equation, predict the value of x if y is 426.21, rounding to the nearest tenth.
[Only an algebraic solution can receive full credit.]


Review the Second Approach: Step-by-step screenshots of how to use the TI-84 Plus to determine Exponential Regression Equation





$ y = ab^x \\[3ex] y \approx 2.19(3.23)^x ...nearest\;\;hundredth \\[5ex] when\;\; y = 426.21 \\[3ex] 426.21 = 2.19(3.23)^x \\[3ex] 2.19(3.23)^x = 426.21 \\[3ex] 3.23^x = \dfrac{426.21}{2.19} \\[5ex] 3.23^x = 194.6164384 \\[3ex] \ln 3.23^x = \ln 194.6164384 \\[3ex] x \ln 3.23 = \ln 194.6164384 \\[3ex] x = \dfrac{\ln 194.6164384}{\ln 3.23} \\[5ex] x = \dfrac{5.271030639}{1.172482137} \\[5ex] x = 4.495617009 \\[3ex] x \approx 4.5 ...nearest\;\;tenth $
(3.) NYSED The table below gives air pressures in kPa at selected altitudes above sea level measured in kilometers.

x Altitude (km) 0 1 2 3 4 5
y Air Pressure (kPa) 101 90 79 70 62 54

Write an exponential regression equation that models these data rounding all values to the nearest thousandth.

Use this equation to algebraically determine the altitude, to the nearest hundredth of a kilometer, when the air pressure is 29 kPa.


Review the Second Approach: Step-by-step screenshots of how to use the TI-84 Plus to determine Exponential Regression Equation





$ y = ab^x \\[3ex] y \approx 101.523(0.883)^x ...nearest\;\;thousandth \\[3ex] when\;\; y = 29 \\[3ex] 29 = 101.523(0.883)^x \\[3ex] 101.523(0.883)^x = 29 \\[3ex] 0.883^x = \dfrac{29}{101.523} \\[5ex] 0.883^x = 0.2856495572 \\[3ex] \ln 0.883^x = \ln 0.2856495572 \\[3ex] x \ln 0.883 = \ln 0.2856495572 \\[3ex] x = \dfrac{\ln 0.2856495572}{\ln 0.883} \\[5ex] x = \dfrac{-1.252989544}{-0.1244300784} \\[5ex] x = 10.06982846 \\[3ex] x \approx 10.07\;km ...nearest\;\;hundredth $
