(9.) A population, P0 doubles every 5 years.
(a.) Find the tripling time of this population.
(b.) Find the percent increase of the population after two tripling periods.


Let:
t = time
P0 = initial population at t = 0
P = population at a specific time
b = base (growth factor)

$ P(t) = P_0 * b^t...\text{exponential growth function} \\[3ex] \text{At } t = 5: \\[3ex] P(5) = P_0 * b^5 \\[3ex] Also,\;\; P(5) = 2 * P_0 ...\text{initial population doubles every 5 years} \\[3ex] P(5) = P(5) \implies \\[3ex] P_0 * b^5 = 2 * P_0 \\[3ex] b^5 = 2 \\[3ex] \text{Take the fifth root of both sides} \\[3ex] b = \sqrt[5]{2} \\[3ex] \text{Hence, the exponential function is:} \\[3ex] P(t) = P_0 * (\sqrt[5]{2})^t \\[3ex] $ The tripling time of the population is the time it takes the population to triple (to be three times the initial population)

$ (a.) \\[3ex] P(t) = 3P_0 ...\text{population is triple the initial population} \\[3ex] \implies \\[3ex] 3P_0 = P_0 * (\sqrt[5]{2})^t \\[3ex] 3 = (\sqrt[5]{2})^t \\[3ex] \text{Introduce logarithm to both sides} \\[3ex] \log 3 = \log (\sqrt[5]{2})^t \\[3ex] \log 3 = t \log \sqrt[5]{2} \\[3ex] t = \dfrac{\log 3}{\log \sqrt[5]{2}} \\[5ex] t = 7.924812504\;years \\[3ex] $ The tripling time of this population is 7.924812504 years.
A tripling period is the time taken for the population to be three times the initial population.
Two tripling periods is the time taken for the population to be nine times (3² = 9) the initial population.

$ (b.) \\[3ex] \text{Initial Population} = P_0 = Initial \\[3ex] \text{Population after 2 trimpling periods} = 9 * P_0 = New \\[3ex] \%\;increase = \dfrac{New - Initial}{Initial} * 100 \\[5ex] = \dfrac{9P_0 - P_0}{P_0} * 100 \\[5ex] = \dfrac{8P_0}{P_0} * 100 \\[5ex] = 800\% \\[3ex] $ The percent increase of the population after two tripling periods is 800%.

An exponential function is defined for all the real values of the independent variable
Therefore, all the real values of $x$ are defined in the domain.

Because all the real values of $x$ are in the domain of an exponential function, an exponential function does not have a vertical asymptote.
